# 🛡 Silent Guard Geosite Rules

Набор правил маршрутизации для клиентов **V2Ray / Xray / Happ**.

Эти правила помогают отправлять часть российских сервисов **напрямую**, без проксирования через зарубежный сервер.  
Это может быть полезно, если некоторые приложения, банки, маркетплейсы, карты или локальные сервисы некорректно работают при иностранном IP.

## 📦 Что внутри

Репозиторий содержит готовые файлы правил для маршрутизации:

- `geosite.dat` — список доменов для маршрутизации по категориям;
- `geoip.dat` — IP-диапазоны для маршрутизации по странам и сетям.

Основная категория для прямого подключения:

```txt
geosite:silentguard-direct
```

---

## 🧭 Для чего это нужно

Некоторые российские сервисы могут некорректно работать при подключении через зарубежный IP-адрес.

Например, проблемы могут встречаться у:

- банковских приложений;
- маркетплейсов;
- карт и навигации;
- такси и доставки;
- государственных и локальных сервисов;
- некоторых рабочих и корпоративных сайтов.

Эти правила позволяют оставить общее подключение через прокси/VPN, но при этом направлять выбранные российские сервисы напрямую.


```txt

```

4. Сохраните настройки и перезапустите клиент.

## ✅ Рекомендуемый режим работы

Рекомендуемая логика маршрутизации:

```txt
Блокировка → Прямое подключение → Прокси
```

То есть:

- заблокированные домены обрабатываются первыми;
- российские сервисы из списка идут напрямую;
- всё остальное отправляется через прокси/VPN.

---

## ⚠️ Важно

Эти правила не гарантируют работу абсолютно всех российских сервисов.

Некоторые приложения могут использовать дополнительные методы проверки:

- проверку IP-адреса;
- собственные DNS-запросы;
- проверку установленного VPN-клиента;
- анализ сетевого интерфейса;
- внутренние антифрод-системы.

Поэтому даже при корректной маршрутизации отдельные приложения всё равно могут работать нестабильно.
